Mini-Photoshop

2020

  • Creative Coding

Mini photoshop is a small drawing tool that I created using the p5js library. Many existing drawing softwares focus on imitating how people use traditional drawing tools in the real world. After learning how to create images using coding, I want to explore more ways of how we use brush tools on electronic devices. It contains some interesting brushes that give unexpected effects. This project also helped me to get familiar with coding and showed me more possibility with image creating and coding.
